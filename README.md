    - –†–µ–≥—É–ª—è—Ç–æ—Ä–Ω–∞—è –Ω–µ–æ–ø—Ä–µ–¥–µ–ª–µ–Ω–Ω–æ—Å—Ç—å –≤ digital-—Å—Ñ–µ—Ä–µ

- –¢—Ä–∏–≥–≥–µ—Ä—ã –Ω–∞ –æ—Å–Ω–æ–≤–µ big data –∏ –∞–ª—å—Ç–µ—Ä–Ω–∞—Ç–∏–≤–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö (sentiment analysis, –≤–µ–±-—Ç—Ä–∞—Ñ–∏–∫, —Å–ø—É—Ç–Ω–∏–∫–æ–≤—ã–µ –¥–∞–Ω–Ω—ã–µ)

**–°–ª–æ–π 5: –ò–Ω—Ç–µ–≥—Ä–∞—Ü–∏—è –Ω–µ—Ñ–∏–Ω–∞–Ω—Å–æ–≤—ã—Ö –¥–∞–Ω–Ω—ã—Ö**
